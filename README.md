# etch-a-sketch

A project to create a simple web-based Etch-a-Sketch. It is a web page with a grid of squares that fill in from white to black when the mouse hovers over them. The user is able to specify the grid size for any integer between 1 and 100. 

Based on these specifications from The Odin Project: https://www.theodinproject.com/lessons/foundations-etch-a-sketch
